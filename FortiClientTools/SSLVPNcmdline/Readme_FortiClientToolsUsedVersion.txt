For testing we used: FortiClientTools_7.2.3.0929
All rights belong to Fortinet.

You have to download the Forticlient tools from Forticlient Website and paste them into this folder.

Details: 
- Downlaod Forticlient tools it from https://support.fortinet.com/Download/FirmwareImages.aspx (you must be logged in with your fortinet account).
- in Dropdown select FortiClient -> Select Tab "Download" -> go to Windows/ v7.00/ 7.2/ 7.2.3/  -> Download FortiClientTools_7.2.3.0929.zip
(It might also work with newer versions.)

Then copy the following files here:

\gitrepo\FortiClientTools\SSLVPNcmdline\x86\FCCryptDLL.dll
\gitrepo\FortiClientTools\SSLVPNcmdline\x86\FortiSSLVPNclient.exe
\gitrepo\FortiClientTools\SSLVPNcmdline\x86\libcrypto-3.dll
\gitrepo\FortiClientTools\SSLVPNcmdline\x86\libssl-3.dll
\gitrepo\FortiClientTools\SSLVPNcmdline\x86\mfc140.dll
\gitrepo\FortiClientTools\SSLVPNcmdline\x86\msvcp140.dll
\gitrepo\FortiClientTools\SSLVPNcmdline\x86\vcruntime140.dll
